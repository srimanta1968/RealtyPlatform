1 · Phase Goal

Phases 1–3 built the back office. Phase 4 closes the loop on the moment of truth — the site visit. Field agents arrive prepared, capture rich outcomes effortlessly, and present intelligent alternatives on the spot. The system stops feeling like CRM and starts feeling like an operating system.

In Scope
Field-first capability
Field App Backend (BFF), React PWA for agents, expanded Media Service, Property Matching Agent, Visit Summary Agent, property intake flow, offline-tolerant capture.
Out of Scope
Deferred
Native iOS/Android apps (PWA suffices), Customer Portal (P5), Concierge (P6).

5 · LLM Agents

Agent · Phase 4

Property Matching

Trigger: lead.qualified, visit.outcome (esp. "wants_alternative"), or /field/match API call.

Logic: kNN vector search over property embeddings → rule re-rank (budget fit, availability, drive-time) → optional LLM re-rank with reasoning.

Output: ranked list with reason per item shown in Field App.

Agent · Phase 4

Visit Summary

Trigger: visit.completed + voice memo or notes ≥N chars.

Logic: Whisper transcribes voice → LLM extracts sentiment, objections, interest level, next step → writes structured outcome.

Guardrail: never auto-closes a lead; agent confirms with one tap (flips source to HUMAN).

11 · Exit Criteria

  • ≥90% of completed visits have structured outcome within 1h
  • ≥40% of "wants alternative" outcomes converted to a follow-up visit
  • Field agent NPS for the app ≥40
  • ≥1 alternative property suggested in ≥70% of relevant visits
  • Acquisition team processes new field intake within 48h SLA
