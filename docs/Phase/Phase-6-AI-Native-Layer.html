1 · Phase Goal

Phases 1–5 built the organism. Phase 6 layers a full AI cortex: Concierge handles inbound 24/7, scoring becomes hybrid LLM+ML, Property Intelligence keeps inventory fresh, Campaign Copy drafts outreach, and NBA upgrades from rules to LLM. No agent makes terminal decisions alone.

In Scope
Full AI cortex
AI Gateway · Concierge Service · Agent Runtime · Advanced Lead Scoring · Property Intelligence · Campaign Copy · LLM-powered NBA · agent observability · review queues.
Out of Scope
Deferred
Fully autonomous closure (humans always close) · outbound voice AI · generative property visualization.

2 · Architecture View

The AI Gateway is the single chokepoint for all model calls — provider-agnostic, with prompt registry, PII scrubbing, cost telemetry, and budgets. Agents run inside the Agent Runtime, subscribed to events, and act through the same APIs operators use.

5 · The Six Agents

P6 · NEW

Concierge

24/7 web + WA. Cites Property facts only. Books slots via Visit availability. Hands off on low confidence or HNI signal.

P6 · UPGRADE

Advanced Scoring

Hybrid ML on structured features + LLM extraction from free-text. Replaces Phase 3 scorer.

P6 · NEW

Property Intelligence

Suggests description improvements, tags, stale-flag, price band hints. All DRAFTs requiring approval.

P6 · NEW

Campaign Copy

Drafts subjects + bodies per segment. Multiple variants for A/B. Marketer approves before send.

P6 · UPGRADE

Next-Best-Action (LLM)

Synthesizes lead history + recent context. Suggests one specific action with timing & channel. Surfaces as task; never auto-executes.

P3 · CONTINUES

Qualification & Matching

Continue from earlier phases. Now use AI Gateway routing. Costs and latency tracked alongside new agents.

9 · Prompt & Model Governance

  • All prompts versioned in registry; rollouts gated
  • Model upgrades tested against golden set before promotion
  • Eval harness with offline regression suite per agent
  • Cost guardrails: per-agent monthly budget with alerts
  • Provider failover (primary → secondary) on API errors
  • PII scrubbing layer in AI Gateway before egress

12 · Exit Criteria

  • ≥40% of inbound web/WA enquiries fully handled by Concierge
  • Concierge handoff context rated useful by presales in ≥85% of handoffs
  • Advanced scoring outperforms Phase 3 on closure prediction (offline eval)
  • Marketer-edit rate on Copy Agent drafts <40%
  • NBA suggestion acceptance rate ≥50%
  • Agent monthly cost within budget; cost-per-qualified-lead trending down
